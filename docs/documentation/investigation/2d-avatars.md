# Feature Spec: 2D Party Avatars at the Desk

> **Status: PLANNED — Not yet designed.**
> This spec is a placeholder. Full design to be done in a future brainstorm session.

## Overview

Visual silhouettes representing the arriving party standing at the desk. Clicking them triggers the size lie accusation (replacing the ticket-based size click from Sub-project 1).

## Rough Scope
- 2D avatar silhouettes (number matches `truePartySize`)
- Clickable as an accusation target for size lies
- Allergy/dietary bubble system (feeds into Sub-project 4: Dietary Mismatch)
